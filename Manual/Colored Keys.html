On the Subject of Colored Keys

Who painted this keypad?

This module features four colored keys with letters on them and a display with the name of a color.

Press the key that has the most conditions met.

Top Left: Top Right:
  • There is a lit MSA indicator
  • There is an unlit CAR indicator
  • There is an odd number of battery holders
  • There is an RJ-45 port
  • The color of this key matches the color of another key
  • The color of this key matches the displayed word
  • The color of this key matches the color of the displayed word
  • The letter on this key matches a letter in the displayed word
  • The letter on this key matches a letter in the serial number
